# Paycheck Budget Tracker

A mobile-first web application designed for people who live paycheck to paycheck. Plan and track your spending per pay period rather than monthly, with clear visibility into bills, categories, and spendable amounts.

## Features

### Core Functionality
- **Paycheck-Based Budgeting**: Budget based on your actual pay schedule (weekly, bi-weekly, semi-monthly, or monthly)
- **Bill Management**: Track recurring bills and see which ones are due before your next paycheck
- **Category Tracking**: Set budget categories (fixed, flexible, savings) and track spending in real-time
- **Spendable Amount**: Always know exactly how much money you have left to spend after bills and savings

### User Experience
- **Modern Financial Brutalism Design**: Dark theme with high contrast for clarity and focus
- **Mobile-First**: Optimized for mobile devices with responsive design
- **Real-Time Updates**: See your budget update immediately as you log spending or mark bills paid
- **Simple Onboarding**: Quick 3-step setup to get started

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Styling**: Tailwind CSS with custom design system
- **UI Components**: Radix UI + shadcn/ui
- **Font**: Space Grotesk
- **Icons**: Lucide React

## Getting Started

1. Sign up or sign in at the landing page
2. Complete the 3-step onboarding process
3. Start tracking your paycheck budget!

## Design Philosophy

This app follows the principle that **every number should be immediately scannable**. The user should be able to glance at their phone and know their financial status in under 3 seconds.
